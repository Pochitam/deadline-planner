# DayLink
Веб-приложение для управления личными задачами и дедлайнами. Предоставляет пользователю интерфейс для создания задач с указанием срока выполнения, отслеживания их статуса и визуальной индикации просроченных дедлайнов.
# Цель проекта
Предоставить простой и удобный инструмент для управления задачами и дедлайнами, доступный любому пользователю без регистрации и лишних настроек.
# Основной функционал
<ul>
  <li>Добавить задачу с названием и дедлайном</li>
  <li>Просмотр списка всех задач</li>
  <li>Отметить задачу выполненной</li>
  <li>Удаление задачи</li>
  <li>Визуальная подсветка просроченных задач</li>
</ul>

<h3>Дополнительно</h3>
<ul>
  <li>Категории (учёба / работа / личное)</li>
  <li>Приоритет (высокий / средний / низкий)</li>
  <li>Описание/заметка к задаче</li>
  <li>Сортировка по дате дедлайна</li>
</ul>

# Технологический стек
<h3>Backend</h3>
<ul>
  <li><b>Python</b></li>
  <li><b>FastAPI</b></li>
  <li><b>SQLite</b> — база данных</li>
  <li><b>SQLAlchemy</b> — ORM для работы с базой данных</li>
  <li><b>Pydantic</b> — валидация данных (встроен в FastAPI)</li>
  <li><b>Uvicorn</b> — ASGI сервер для запуска приложения</li>
  <li><b>python-dotenv</b> — переменные окружения</li>
</ul>

<h3>Frontend</h3>
<ul>
  <li><b>HTML</b></li>
  <li><b>CSS</b></li>
  <li><b>JavaScript (vanilla)</b> — fetch запросы к API, динамическая отрисовка</li>
</ul>

<h3>Взаимодействие компонентов</h3>
<p>
  Пользователь работает с интерфейсом в браузере. Фронтенд (HTML/CSS/JS) 
  отправляет fetch-запросы к FastAPI. FastAPI обрабатывает запросы, валидирует 
  данные через Pydantic и обращается к базе данных SQLite через ORM SQLAlchemy. 
  Ответ возвращается в формате JSON и отрисовывается на странице без перезагрузки.
</p>

<pre>Браузер (HTML/CSS/JS) → fetch запросы → FastAPI → SQLAlchemy → SQLite</pre>
